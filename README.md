### Install dependencies
```
$ npm install # Or $ yarn install
```

### Setup file .env 
```
$ cp .env.example .env
```

### Build for production
```
$ npm run build
```

### Build for development with hot reload
```
$ npm run start
```

### Run test
```
$ npm run test
```
